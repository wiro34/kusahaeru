くさはえる
==========

## Demo

* [here](http://wiro34.github.io/kusahaeru-demo/)

## Requirements

* Node >= v0.10.28
* Gulp >= 3.8.10
* Bower >= 1.3.12

```
npm install -g gulp bower
```

## Development

```
git clone https://github.com/wiro34/kusahaeru.git
cd kusahaeru
npm install
bower install
gulp serve
```
